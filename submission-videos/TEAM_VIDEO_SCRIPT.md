Hi — I’m Saadia Asghar, Team HN-6590, and I built BenchDial solo for The Negotiator challenge with ElevenLabs and Hack-Nation.

BenchDial is a voice agent system for laboratory equipment repair. When a centrifuge goes down, labs get opaque phone quotes that hide call-out fees, calibration, and warranty. I built the full loop the challenge asks for:

Estimator — voice interview plus a service report into one locked ScopePrint.  
Caller — three negotiation styles with the same job every time.  
Closer — verified leverage, a thirty percent red-flag rule, and a ranked award memo with transcript evidence.

I handled product, agent prompts, full-stack, and the honesty firewall myself. BenchDial recommends; a human still decides.

Thanks to ElevenLabs, Hack-Nation, and the MIT clubs.

### SHOT LIST (simple)

1. 0–5s — Face on camera, smile, introduce name + solo
2. 5–35s — Stay on camera (or screen-share Home behind you)
3. 35–50s — Say Estimator / Caller / Closer quickly
4. 50–55s — Close: “human still decides” + thank you

### TIPS

- Speak slightly slower than normal; smile once
- Export as H.264 MP4 (not HEVC)
- Blur any passwords / API keys if a screen is visible
- If nervous: print this page and read — judges care about clarity, not polish
